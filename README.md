# Heróis dos Elementos

Este é um jogo de storytelling interativo feito em Python.

## História
Os jogadores são os Heróis dos Elementos e precisam entrar em um templo mágico para recuperar a Pedra dos Elementos antes que o templo desmorone.

## Como jogar
Execute o arquivo `main.py` no terminal:

python main.py

## Estrutura do projeto
- main.py → controla o jogo e as escolhas
- historia.py → contém os textos da história
